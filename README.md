# 🏥 Clínica Saúde Total - Website

Este projeto é um site simples desenvolvido em HTML e CSS para representar uma clínica médica chamada **Saúde Total**.

## 📄 Estrutura do Projeto

O site é composto por duas páginas principais:

* **index.html** → Página inicial da clínica
* **servicos.html** → Página com a lista de serviços oferecidos
* **style.css** → Arquivo de estilos (CSS)

---

## 🎯 Objetivo

O objetivo deste projeto é praticar conceitos básicos de desenvolvimento web, como:

* Estruturação de páginas com HTML
* Uso de tags semânticas (`header`, `main`, `section`, `footer`)
* Navegação entre páginas
* Aplicação de estilos com CSS

---

## 🧩 Funcionalidades

✔ Página inicial com apresentação da clínica
✔ Link para navegação entre páginas
✔ Página de serviços com lista simples
✔ Layout limpo e organizado

---

## 🚀 Como executar

1. Baixe ou clone o projeto
2. Abra o arquivo `index.html` no navegador
3. Navegue pelo site usando os links
